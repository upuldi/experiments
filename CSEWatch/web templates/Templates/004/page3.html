Navigation Bar

You can easily add a third row of links to the top menu bar if you have more than ten web pages in your website.

You will need to increase the height of the second row (Now 64) on the index.html web page to accomodate the change.

You'll also need to increase the height of the topnavigate division in the style sheet.

Add the links with page names and urls for the third row to navigation.html.

You could also increase the width of the center frame in the navigation row and add scrolling if desired. It looked a bit tacky to me so I left it non-scrolling.
