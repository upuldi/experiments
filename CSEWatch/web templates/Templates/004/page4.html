CSS - Style Sheet

Font settings in the content pages can be changed by altering the framesset.css file located in your zip file download.

You can change the font family of choice, the font size, paragraph text alignment and more.

Changes to the CSS file can be made using a simple text editor like NotePad.

Button and Background Colors

To change the buttons, you must change button names and the size of the buttons in the style sheet.

The background colors of different sections of the web page can be alterd by changing the background-color attribute for the respective division in the style sheet.
