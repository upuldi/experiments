Your Goal

Your goal in learning to build a website should be to progress to liquid designs.

Start with the simplest fixed width designs and learn to modify them first.

Then learn to add the navbar menus by studying lists and the templates that use them.

If you want to use the round cornered dialogue boxes learn how to add them.

You can copy the HTML code from other templates and also the sections from the style sheets that set up the added features.

Get ahead of the game and learn the right way of building web pages.

Contact Form

This contact form is not active.

To activate it change the action attribute on the form (contact.html) to action="feedback-form.php".

When you publish your website, upload the feedback-form.php script (located in your zip file) to the same folder where this page will be located.

Be sure to replace email@address on the form in 3 locations with your own email account information before uploading. You must create your email accounts before you test the form.

Change the file permissions on the script to 755 using the CHMOD function of your publishing tool. This is done after you upload the altered script.

We recommend the use of Google reCaptcha on all forms you place online.

Replace these instructions with your own contact form information!
