Why Go Liquid?

The majority of websites built these days use static or fixed width structures.

Most of the fixed width pages on those websites are set to 1000 pixels or wider.

Every year we see new devices sent to the marketplace that can be used to browse websites. Tablets, smaller and smaller notebooks and other devices appear all the time.

Some of these new devices have screens as small as seven inches and viewing resolutions as low as 800x600.

A visitor viewing a static web page on a device with an 800x600 viewing resolution would have to scroll 200 pixels to the right to see all of the static page.

Properly designed liquid web pages would conform to the visitor's device and require no scrolling to view.

Side Margins

We added the side margins to these designs to keep the page from spreading out too much at higher resolutions.

The side margins can be adjusted or removed at the top of your style sheet.

Keep It Simple

A lot of folks get carried away these days when building websites.

They add so many gadgets to the design that the main purpose of the website is lost in the foray.

The main focus of any website should be its content. Simple design always out performs the gadget website.

Testing Your Work

As with any website you should test your website on as many different browsers as possible.

When creating liquid websites you also want to test your website at different monitor resolutions.

You should also get into the habit of checking your HTML code and CSS at the W3C Validation Service.
