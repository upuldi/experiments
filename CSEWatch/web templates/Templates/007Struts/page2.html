More Information

You can create a website using this frames template and a free HTML editor from our collection.

1..Replace the text on each web page with your own.

2..Rename the web page that contains your text.

3..Change the navigation.html web page to reflect the change in page name.

Replacing the Image

For best results replace the image in the heading with an image of the same width. The existing image is 1000 pixels in width. The height of the image is defined in the style sheet. 20% of your visitors still use a 1024X768 monitor resolution. If you use a wider image, those visitors will need to do some right scrolling to view your heading.

More on Redirection

When the individual pages are redirected they go to the main frameset page (index.html) and the page that is directly accessed appears in the content window.

Heading Text

If you don't like the way the heading text moves around the page at different browser widths and resolutions, get a copy of the Free Photo editor PhotoScape and add the text to the photo.
