More Information

You can create a website using this frames template and a free HTML editor from our collection.

1..Replace the text on each web page with your own.

2..Rename the web page that contains your text.

3..Change the navigation.html web page to reflect the change in page name.

Replacing the Image

For best results replace the image in the heading with an image of the same width. The existing image is 400 pixels in width. The height of the image is defined in the style sheet, but should be close to 600-650 pixels.

Monitor Resolution

20% of your visitors still use a 1024X768 monitor resolution. If you add buttons or a taller image, be sure to test the template at 1024x768.

More on Redirection

When the individual pages are redirected they go to the main frameset page (index.html) and the page that is directly accessed will appear in the content window.

Be sure to leave the javascript in place on all pages of the website.
