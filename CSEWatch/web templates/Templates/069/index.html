Your Goal

Your goal in learning to build a website should be to progress to liquid designs.

Start with the simplest fixed width designs and learn to modify them first.

Then learn to add the navbar menus by studying lists and the templates that use them.

If you want to use the round cornered dialogue boxes learn how to add them.

You can copy the HTML code from other templates and also the sections from the style sheets that set up the added features.

Get ahead of the game and learn the right way of building web pages.

Linked Style Sheet

This web page uses a linked style sheet.

It can be edited with any simple text editor.

It controls things like font size, color, weight and alignment.

Linked style sheets reduce the size of your web page and make your web site easier to manage.
