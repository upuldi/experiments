Your Goal

Your goal in learning to build a website should be to progress to liquid designs.

Start with the simplest fixed width designs and learn to modify them first.

Then learn to add the navbar menus by studying lists and the templates that use them.

If you want to use the round cornered dialogue boxes learn how to add them.

You can copy the HTML code from other templates and also the sections from the style sheets that set up the added features.

Get ahead of the game and learn the right way of building web pages.

Liquid Designs

This template uses a liquid design. The web page will expand and contract with the width of the visiting browser.

It will also adjust to the various resolutions of various viewing devices.

Test this web page at a monitor resolution of 800x600 if you can. This is how your website will be viewed on some tablets. Compare it to some of the fixed with templates.

Heading Background Image

The heading image used in this document is about 1000 pixels in width. If you want your heading to display full width at 1920 pixels, you'll need to replace it with an image of that width. Then adjust the height of the head division to that of your new image.

Another option for liquid web pages is to omit the background image or use small texture type background images that repeat and fill the heading area.
